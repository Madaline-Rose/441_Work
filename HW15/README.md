# Reflection

Working on this project taught me that the most important part of three.js is understanding that everything exists in a true 3D coordinate space (x, y, z), not just a flat surface. Unlike 2D tools like Canvas or Phaser where depth is mostly simulated, in three.js objects actually exist in space, which means camera position, perspective, and depth all directly affect what you see. Even in small changes in z-position or camera angle can completely change the composition, which made me think more like I was placing objects in a physical enviroment rather then a drawing screen. I have worked with Blender and although the process is different I did feel I had to think similar in regards to these aspects. 

The most challenging part of this project was getting the GLB model to load properly and making everything feel visually balanced in 3D space. At first, I had issues where the model either didn’t appear or looked too large/small compared to the shapes. I worked through this by adjusting scale values, experimenting with object positions, and testing changes ones step at a time using Live Server. I did use Gen AI (Chatgpt) to help debug structure issues and my GLB model issue. I also used it when I got a little stuck on animation loops. It helped me quickly identify problems in my setup and improve my scene design decisions. 

The intention behind my scene was to create a small, playful “dream state” where a cat imagines food floating around it. I wanted the experience to feel slightly calm but funny. The donut and apple orbiting the cat help reinforce that idea, while the subtle movement in both the food and the cat makes the scene feel alive rather than static. 

If I had more time, I would improve the lighting to create more depth and possibly add a soft glow or particle effects to make the food feel more like glowing thoughts. I’m also interested in exploring camera controls like OrbitControls and learning how to create more advanced animations, especially smoother motion transitions and interactive scenes where the user can affect the 3D environment.
